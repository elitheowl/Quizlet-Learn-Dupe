## 📥 **Installation**

### 🚀 **Option 1: Quick Install (Recommended)**

1. **📥 Download** the latest release from [Releases](https://github.com/elitheowl/Quizlet-Learn-Dupe/releases/latest)
2. **📦 Extract** the `Quizlet-App-Release-v1.0.0.zip` file
3. **⚡ Run** `Install-Quizlet-App.bat` for automatic setup
4. **🎉 Enjoy** - Desktop shortcuts created automatically!

### 💻 **Option 2: Portable Use**

- Simply **double-click** `quizlet-app.exe` from any location
- Perfect for **USB drives** and **portable installations**
- **No installation required** - runs from anywhere!

### 🔧 **Option 3: Manual Installation**

1. Extract to a permanent folder (e.g., `C:\Program Files\Quizlet App\`)
2. Create shortcuts manually if desired
3. Run `quizlet-app.exe` directly

## 🛠️ **Development**

### 🔧 **Prerequisites**

- [Node.js](https://nodejs.org/) (v16 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### 📦 **Install Dependencies**

```bash
npm install
```

### 🚀 **Run Development Server**

```bash
npm run dev
```

### 🏗️ **Build for Production**

```bash
npm run build
```

### 📱 **Build for Different Platforms**

```bash
# Windows 64-bit
npm run build:win64

# Windows 32-bit
npm run build:win32

# All Windows versions
npm run build:win
```
